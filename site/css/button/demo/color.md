# 按钮颜色

- order: 1

不同颜色的按钮。

---

````jsx
var Button = require('uxcore-button').Button;

React.render(
	<div>
		<Button color="blue">default blue</Button>
		<Button color="orange">orange</Button>
		<Button color="white">white</Button>
	</div>
, document.getElementById('css-button-demo-color'));
````
