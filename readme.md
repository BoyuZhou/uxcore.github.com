# uxcore

uxcore demo站点，模板生成采用 nico + nico-jsx + jade

## Usage

### 初始化
```sh
$ git clone https://github.com/uxcore/uxcore.git
$ cd uxcore.github.io
& npm install -g nico jade
$ npm install
$ git submodule init
$ git submodule update
```
### 本地开发
```sh
$ npm run start
```

### gh-pages部署
```sh
$ npm run deploy
```
